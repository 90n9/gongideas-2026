ช่วงนี้ใครอยากเริ่มเรียน AI หรืออัปเดต skill ด้านนี้ — มีอีกเว็บที่ผมอยากแนะนำคือ **[learn.deeplearning.ai](https://learn.deeplearning.ai/)**

## คอร์สอะไร

เป็น learning platform ของ **Andrew Ng** (อาจารย์ดังด้าน AI/ML จาก Stanford ผู้ก่อตั้ง Coursera และ deeplearning.ai) ร่วมกับพาร์ตเนอร์เจ้าใหญ่ในวงการ — OpenAI, Anthropic, LangChain, Meta, Mistral ฯลฯ

จุดเด่นคือ:

- **คอร์สสั้น** — ความยาวประมาณ **1 ชั่วโมง** ต่อคอร์ส เรียนจบในนั่งเดียว
- **ฟรีเกือบทั้งหมด** — short course ส่วนใหญ่ฟรี
- **hands-on** — มี Jupyter notebook ให้รัน code จริงในเบราว์เซอร์ ไม่ต้องเซตอัพอะไร
- **อัปเดตเร็ว** — เรื่องใหม่ ๆ มาเร็วมาก (RAG, agents, MCP, fine-tuning)

## ตัวอย่างหัวข้อที่มี

- **LLM Application Development** — สร้าง chatbot, RAG, function calling
- **Agents** — multi-agent system, tool use
- **Fine-tuning** — train โมเดลให้เก่งงานเฉพาะทาง
- **Prompt engineering** — เขียน prompt ให้ได้ผลตามที่อยากได้
- **MCP** — Model Context Protocol ของ Anthropic
- **Vector database / embeddings** — สำหรับงาน RAG หรือ search

## ทำไมแนะนำ

- เรียนพร้อม **เขียนโค้ดจริง** ไม่ใช่นั่งฟังทฤษฎีอย่างเดียว
- หัวข้อ **practical** — เอาไปต่อยอดกับ project จริงได้เลย
- ฟรี + สั้น = **ลองได้ไม่กี่คอร์ส** ก็พอจะเริ่มทำงานเองเป็น

## เริ่มยังไง

เข้าไปที่ [learn.deeplearning.ai](https://learn.deeplearning.ai/) สมัครด้วย email แล้วเลือกคอร์สที่สนใจ — ถ้าไม่รู้จะเริ่มจากตัวไหน ลอง **"ChatGPT Prompt Engineering for Developers"** ก่อนได้ครับ เป็นตัวคลาสสิกที่หลายคนเริ่มจากตรงนี้
